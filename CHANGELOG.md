# Changelog

All notable changes to the **AI Engineering Knowledge System** will be documented in this file.

This project follows the principles of **Semantic Versioning (SemVer)** and maintains a chronological history of significant updates.

---

# Versioning Scheme

- **Major (`X.0.0`)** — Significant restructuring or major new editions.
- **Minor (`0.X.0`)** — New volumes, chapters, or major features.
- **Patch (`0.0.X`)** — Documentation improvements, corrections, bug fixes, formatting updates, and small enhancements.

Example:

```
v1.0.0
│ │ └── Patch
│ └──── Minor
└────── Major
```
